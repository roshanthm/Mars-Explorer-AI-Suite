Mars Explorer AI Suite is an interactive, AI-powered web application designed to bring the wonders of Mars directly to users.
Built with Streamlit, powered by Groq Llama 3, and enriched with real NASA APIs, it blends science, data, and play into one seamless exploration experience.

This platform reimagines Mars discovery through:

🧠 An intelligent Mars Chatbot

📘 A Mars Knowledge Center

🛰 NASA data dashboards with real rover images

🎮 A Space Explorer Quiz with generative AI questions

🎨 Smooth UI, animations, and modular navigation

Everything is crafted for curiosity, learning, and inspiration.
